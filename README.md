# raashishah.com

Personal portfolio built with Next.js App Router, Vitest, and Playwright.

## Development

```bash
npm install
npm run dev        # http://localhost:3000 (or 3002 if port is taken)
npm test           # Vitest unit tests
npm run test:e2e   # Playwright end-to-end tests
npm run build      # Production build
npm run lint       # ESLint
```

## Deploy

Push to `main` deploys to Vercel (production).
